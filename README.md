# 春节红包活动提醒网站

这是一个帮助您在春节期间收集红包活动信息的网站，包含主流平台的红包活动和详细教程。

## 功能特点

- ⏰ **春节倒计时**：实时显示距离春节还有多长时间
- 🔥 **热门活动**：展示6大主流平台的红包活动
  - 腾讯元宝分10亿
  - 支付宝集五福
  - 抖音春节红包雨
  - 快手春节大红包
  - 京东年货节红包
  - 拼多多现金大放送
- 📖 **详细教程**：每个活动都有完整的参与步骤和小贴士
- 📱 **响应式设计**：支持手机、平板和电脑访问

## 如何使用

### 方法一：直接打开网页
1. 下载或克隆此仓库
2. 直接在浏览器中打开 `index.html` 文件

### 方法二：使用本地服务器
```bash
# 在项目目录下运行
python3 -m http.server 8080
# 然后在浏览器访问 http://localhost:8080
```
